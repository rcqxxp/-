組別號碼：17


系級班級：資工1A


成員資訊：陳品叡、謝宜達


小專題題目：選機不踩雷


程式介紹：

1.提供一個手機資料庫

2.根據使用者輸入條件進行篩選

3.顯示符合需求的手機清單

4.選出想要的手機進行對照


程式如何安裝執行：

1.在github的download下載檔案

2.將檔案解壓縮

3.在Visual Sudio打開資料夾

4.開啟後執行
